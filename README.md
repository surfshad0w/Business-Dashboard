# Business Income Dashboard Prototype

Simple static dashboard prototype for a business owner who wants one-page visibility into daily and monthly income across multiple unrelated businesses.

## Included business examples
- Shine & Lube Express, car wash with lube
- Oak Barrel Liquor, liquor store
- Sunset Stay Motel, motel
- Fresh Spin Laundry, laundromat
- Summit Rental Property Group, rental property

## Features
- Daily and monthly income cards per business
- Combined revenue summary
- Pie chart showing today's revenue mix
- Simple layout designed to be easy to host on GitHub Pages
- Clear space for future expense tracking enhancements

## Run locally
Open `index.html` in a browser.

## Publish on GitHub Pages
1. Push this folder to a GitHub repository.
2. In GitHub, open Settings > Pages.
3. Set deployment source to the main branch root, or `/docs` if you move the files there.
4. Save and wait for the site URL.
